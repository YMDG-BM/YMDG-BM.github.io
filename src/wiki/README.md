如果说**学习笔记**是探索途中尚未完工的脚手架——那里允许断裂、允许跳跃、允许个人化的试错与涂改——那么 **Wiki** 便是从这片工地里生长出来的、经过反复敲打与校验的基石与梁柱。

这里存放的不是一时的灵感或零散的随想，而是从学习笔记中**升华与凝结**后的确定性知识。每一个概念都被重新审视、拆解、关联，最终安放在一张清晰而自洽的网络之中。它不追逐瞬时的手感，而是追求长期可信的参照；不保留试错的痕迹，而是呈现经过验证的结构。

## 🔗 与学习笔记的关系

- **学习笔记**：过程导向，允许不完整、允许个人化表达，是知识的“毛坯”与“生长点”。
- **Wiki**：结果导向，追求稳定、可检索、可复用，是知识的“成品”与“地标”。

当你在学习笔记中反复验证某个概念、理清某条脉络之后，它就有可能被“升维”写入 Wiki。反过来，Wiki 也为学习笔记提供了可依赖的锚点——当你迷失在细节中时，可以先回到 Wiki 找回地图。

## 🚧 关于 WIP

虽然 Wiki 追求确定性，但它同样不是一蹴而就的。当前仍有大量条目处于待整理、待补充、待关联的状态。标记 `(WIP)` 既是诚实的自省，也是邀请——如果你发现某个概念值得被沉淀，欢迎通过 Issues 或 PR 提出建议。
